BEE




Integrado com inteligência artificial, ‘Bee’ é capaz de recolher, tratar, analisar e retornar dados aos usuários do sistema. Pensando no problema de Análise de Dados da SANOFI, ‘Bee’ poderia mostrar os dados em formato de dashboard, centralizando os dados de todas as compras que estivessem agrupadas em um banco de dados. Indo um pouco além, pensamos até em um ChatBot do ‘Bee’, onde o usuário poderá conversar com a IA para saber sobre certas compras ou até mesmo uma conversa com opções para o usuário selecionar, e assim, conduzindo um certo diálogo com o bot. Isso trará uma facilidade para o colaborador de forma gigantesca, visto que, todas as informações estariam no ‘Bee’, e bastaria algumas perguntas para que ele retornasse uma tabela mostrando as colunas que o usuário gostaria de ver de tal evento de tal data.
